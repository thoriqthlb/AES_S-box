# AES S-Box

AES S-Box modification using 8x8 matrices, equipped with comprehensive cryptography testing.

## File Structure

```
project/
│
├── app.py              # Main App
├── sbox_core.py        # Main function
├── crypto_tests.py     # Cryptography Testing
├── encryption.py       # Encrypt / Decrypt
├── requirements.txt    # Python Dependencies 
└── README.md           # Documentation
```

## Instalasi

### 1. Install Virtual Environtment

```bash
python -m venv venv
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Run

```bash
venv\Scripts\activate
python app.py
```

Localhost : `http://127.0.0.1:5000`
